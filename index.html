MSIT DASHBOARD


Main intention of this project is to make required data accessible for the students. Especisally students can see their attendance which is extrated based on zoom calls. Course grades it includes IT and Soft Skills data. Moreover, we can display our performance in our curriculum by using graphical representation (Line and Bar graph).




About React tool


In our project we have used React.js as a front end technology. We implemented the code in React App. We have used different components like Jumbotron, Accordian, Tbles, Tabs and many more... Initially we have learnt React through various online resources and started working in it. Moreover, we implemented all graph representation either line graph or bar graph using the Chart.js.

Sprint 2


First and foremost, I began working with the component Jumbotron and made it responsive. After it was successfully implemented, I replicated it on the remaining pages, namely the Attendance page, Presentation page, and Transcript page. One of the difficulties I encountered while implementing this component was that there were a lot of empty spaces in the web version, which took some time to fix. Sprint 5

Sprint 8


The main goal of this week was to collect the feedback from the sample. Here we decided to take samples from students of IIIT-Hyderabad and JNTUH learning centers and IT and Soft Skill Mentors. So, we started with designing the questions which needs to be ask for the respondents. Finally, we came up with 10 questions that needs to be ask for the respondents. We have collected the data of the respondents and recorded the session to capture the user experience while using the dashboard. The key takeaways from the respondents are to add few more colors for some of the pages to make the User Interface more vibrant. To add Toefl scores of SoftSkills so that they can understand their performance. Also to improve the search bar which is displaying in the Mentor's Dashbaord. Therefore these are some of the improvements which were requested from the students.
